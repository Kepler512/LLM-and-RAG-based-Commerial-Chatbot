A high-performance Retrieval-Augmented Generation (RAG) system designed to interactively explore the "Diffuse to Choose" (DTC) Virtual Try-All architecture. Engineered with a Pure-LCEL LangChain backend, Google's Gemini 2.5 Flash, and local Chroma vector storage to eliminate LLM hallucinations. This project provides accurately sourced, context-aware technical answers through a modern, responsive Streamlit dashboard featuring custom CSS and real-time document verification.
